  • Untuk menggunakan fitur penggambaran (line, square, rectangle, polygon), silahkan pilih bentuk yang ingin dibuat, dapat dilihat di menu bagian bawah.
  • Untuk memilih warna, silahkan pilih warna yang ingin dibuat, dapat dilihat di menu bagian bawah (color picker).
  • Kalian dapat memilih opsi "fill" untuk menggambar objek yang solid
  • Kalian dapat save gambar yang telah dibuat dengan menekan tombol "save", file akan disimpan di direktori Downloads dalam bentuk json
  • Kalian dapat mengimport/load file json yang telah disimpan sebelumnya dengan menekan tombol "load", file akan langsung ditampilkan di layar
